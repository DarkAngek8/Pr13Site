Що таке GNU/Linux?

GNU/Linux — це вільна й відкрита операційна система, що поєднує ядро Linux, розроблене Лінусом Торвальдсом у 1991 році, та інструменти проєкту GNU. Вона є основою для тисяч дистрибутивів, які використовуються у серверах, десктопах, мобільних пристроях та навіть у вбудованих системах.

Основні риси Linux:

  • Відкрите програмне забезпечення (Open Source).
  • Висока безпека та надійність.
  • Можливість використання на різних пристроях.
  • Величезна спільнота розробників.

Популярні дистрибутиви Linux

  • Ubuntu — один з найпопулярніших дистрибутивів для початківців.
  • Debian — стабільний та універсальний дистрибутив.
  • Fedora — сучасний дистрибутив із новими технологіями.
  • Arch Linux — мінімалістичний, для досвідчених користувачів.
  • Linux Mint — зручний дистрибутив для переходу з Windows.
  • Red Hat Enterprise Linux (RHEL) — корпоративне рішення.
  • Kali Linux — спеціалізований для кібербезпеки.

Основні команди Linux CLI

pwd — показати поточний каталог.
echo "Hello" — вивести повідомлення.
ls — показати список файлів.
cd /path — змінити каталог.
touch file.txt — створити порожній файл.
mkdir new_folder — створити папку.
cp file1 file2 — копіювати файл.
mv old new — перемістити/перейменувати файл.
rm file.txt — видалити файл.
history — показати історію команд.
